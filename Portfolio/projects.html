Project 01  /  Nutrition Platform

NutriTrack

Nutrition & Recipe Platform

NutriTrack is a dark-themed, multi-page nutrition platform and recreation of a past school project — rebuilt with a sharper design system and interactive components. It helps users reach their health goals through high-protein recipes like Chicken Parmesan and Salmon Pasta, with each dish featuring full ingredients, step-by-step instructions, and macro breakdowns to help users unlock their potential.

Dynamic Recipe Switcher — Sticky thumbnail bar that swaps all recipe content without a page reload, driven by URL params.
Interactive Stepper & Timers — Click any step to activate it with a green glow, with built-in countdown timers on cooking steps.
Adjustable Servings — Counter that recalculates all nutrition values proportionally in real time.
Filter Tabs & Macro Tracking — Filter recipes by category and view animated macro progress bars for each meal.
Rebuilding from scratch — Required a complete redesign of architecture, styles, and JavaScript logic over the original school project.
Dynamic content rendering — Building a single detail page that dynamically renders all six recipes from a JS data object — including timers, steppers, and nutrition — required careful state management in vanilla JS.
Consistent dark design system — Maintaining a cohesive aesthetic across four unique page layouts while keeping CSS variables and components perfectly aligned.
HTML5 CSS3 JavaScript

Project 02  /  E-Commerce

Velour

Luxury Perfume E-Commerce Website

Velour is a sophisticated e-commerce platform designed to provide a premium online shopping experience for fragrance enthusiasts. The website features a sleek, dark-themed aesthetic that mirrors the elegance of the luxury perfumes it showcases — blending gold-on-black styling with immersive product storytelling.

Curated Collections — Organizes high-end fragrances into distinct brand lineups including Armani, Jean Paul Gaultier, and Creed.
Dynamic Shopping Experience — User-friendly navigation, dedicated shop section with PHP pricing, and an integrated cart system.
Brand Engagement — Compelling landing page with high-quality product imagery and descriptive intensity profiles.
Account Management — Dedicated portals for user accounts, order tracking, and direct contact options.
Learning PHP on the fly — Velour was built when PHP was still largely unfamiliar territory. Managing server-side logic, session handling, and dynamic pricing calculations required a steep and fast learning curve mid-project.
Cart & pricing integration — Wiring a functional cart system with PHP-driven pricing while keeping the frontend consistent and bug-free was one of the most demanding parts of the build.
Shipped regardless — Despite the unfamiliarity with the stack, the project was completed and delivered — a testament to persistence and learning under pressure.
HTML5 CSS3 JavaScript PHP

Project 03  /  E-Commerce

ScentSational

Boutique Perfume E-Commerce Platform

ScentSational is a modern, high-end e-commerce website designed with a focus on ethical luxury and a personalized user journey. The platform moves away from traditional dark luxury themes toward a clean, airy, and inviting boutique aesthetic — crafted for the conscious consumer who values sustainability and individuality.

Signature Scent Discovery — Welcoming hero that encourages users to find a fragrance tailored to their personality and occasion.
Ethical Brand Identity — Highlights Sustainable Ingredients, Cruelty-Free practices, and Long-Lasting formulations.
Customer Social Proof — Dedicated testimonial section to build brand trust and community engagement.
Interactive Map & Contact — Fully integrated Google Maps display to locate physical boutique branches, paired with a functional contact form.
Vue.js as a new framework — ScentSational was the first project built using Vue.js. Adapting to a component-based reactive architecture after working with vanilla JS required significant adjustment and self-directed learning.
PHP integration again — Similar to Velour, backend tasks still relied on PHP — managing it alongside Vue's reactivity model added another layer of complexity to the stack.
Balancing design with unfamiliar tooling — Keeping the boutique aesthetic polished and intentional while simultaneously learning the framework meant constantly context-switching between design decisions and technical problem-solving.
Vue.js HTML5 CSS3 JavaScript
